# 双链表
为了克服单链表的访问前驱节点时间复杂度高的问题，引入了双链表，即一个结点内存在前驱指针域 $prior$ 、数据域 $data$ 和后继指针域 $next$ 。

## 双链表的结点
```python
class DLNode:
    def __init__(self, data=None):
        self.data = data
        self.prior = None
        self.next = None
```
## 双链表的实现
```python
class DLinkList:
    def __init__(self, node=None):
        self.head = DLNode()
        self.head.next = None
        self.head.prior = None
```
